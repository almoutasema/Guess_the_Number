# Gissa Talet 🎯

Detta är ett enkelt C#-spel där du ska gissa ett slumpmässigt nummer mellan 0 och 1000.

## 🔧 Hur man kör spelet:
1. Kör programmet.
2. Ange ett nummer mellan 0 och 1000.
3. Få feedback om din gissning är för låg, för hög eller rätt.

## 📌 Funktioner:
- Genererar ett slumpmässigt nummer.
- Validerar användarens inmatning.
- Räknar antal försök.

## 📜 Exempel på användning:




Gissa ett tal mellan 0 och 1000:

500 För lågt! Försök igen.

750 För högt! Försök igen.

625 Rätt svar! Du gissade rätt på 3 försök.
